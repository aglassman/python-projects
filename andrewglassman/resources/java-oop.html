Class Materials

Have these free tools downloaded and installed prior to the class.

  • The Java JDK (Java Development Kit)

    Download and install the Java Development Kit, you'll need version 1.6 or above. Be sure to download the install file that matches your computer platform.

    Download the 1.6 JDK Here.

  • You will also need to install the Eclipse IDE (after JDK is installed). An IDE is an Integrated Development Environment. This is the tool you will use to write, and run your Java Code.

    Download the Eclipse Indigo Release Here.

  • Download the Example Project

    Will be available 6/12/2012

    Unzip this download into your eclipse workspace folder. This should be in your user folder. Next, start eclipse, and go to the workbench. Click File - Import. Under General, select 'Existing Projects into Workspace'. Select 'browse', and navigate to your workspace folder. Select the oop-example directory, and press 'ok'. The project should now appear in your workspace 'project explorer' tab!

  • Download the Power Point presentation if you'd like a copy to follow along with.
